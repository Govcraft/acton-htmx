Security

Your OAuth2 connections are protected with industry-standard security:

  • CSRF Protection: State tokens prevent cross-site request forgery
  • PKCE: Proof Key for Code Exchange prevents code interception
  • Secure Sessions: HTTP-only cookies protect your session
  • Account Linking: Link multiple providers to one account securely
